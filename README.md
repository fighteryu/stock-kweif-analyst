# 科威夫股票技术分析Skill

## 概述

本Skill提供A股和港股的技术分析服务，核心方法论融合了经典量价分析（Wyckoff Method）和点图法（Point and Figure Chart），给出明确的交易建议和目标价。

## 触发条件

用户提及以下内容时使用本Skill：

- "分析 XXX 股票"
- "XXX 股怎么样"
- "股票代码/名称 + 建议"
- "加仓/减仓/止损"
- "目标价"
- "可以买吗"
- "要不要卖"

## 分析流程

### 第一步：获取股票数据

1. **确定股票代码**

   - A股：6位数字代码（如 600519）
   - 港股：5位数字代码（如 00700）
2. **获取历史数据**

   - 使用数据源获取近60个交易日的日线数据
   - 需要字段：日期、开盘价、最高价、最低价、收盘价、成交量

### 第二步：量价分析（核心方法）

#### 2.1 趋势判断

```
上涨趋势：收盘价 > 20日均线 > 60日均线，且20日均线上行
下跌趋势：收盘价 < 20日均线 < 60日均线，且20日均线下行
震荡趋势：价格围绕20日均线波动，均线走平
```

#### 2.2 量价关系分析（Wyckoff核心）

```
1. 放量上涨 = 健康上涨（量价齐升）
2. 缩量上涨 = 顶背离预警（量价背离）
3. 放量下跌 = 可能最后一跌（恐慌抛售）
4. 缩量下跌 = 趋势延续（无人接盘）
5. 放量不涨 = 主力出货
6. 缩量不跌 = 底部信号
```

#### 2.3 关键量价信号

```
买入信号：
- 放量突破20日均线 + 成交量 > 20日均量1.5倍
- 缩量回调至20日均线企稳 + 成交量萎缩至20日均量50%以下
- 底部放量长下影线（锤子线）
- 连续3天成交量放大且股价上涨

卖出信号：
- 放量跌破20日均线
- 缩量上涨至前高附近遇阻
- 高位放量长上影线（射击之星）
- 连续3天成交量放大但股价下跌
```

### 第三步：仓位管理建议

根据分析结果给出明确建议：

| 趋势     | 量价信号       | 建议                             |
| -------- | -------------- | -------------------------------- |
| 上涨趋势 | 放量上涨       | **加仓**（可加20-30%仓位） |
| 上涨趋势 | 缩量上涨       | **持有观察**（警惕顶背离） |
| 下跌趋势 | 放量下跌       | **止损减仓**（建议减50%）  |
| 下跌趋势 | 缩量下跌       | **观望**（不抄底）         |
| 震荡趋势 | 缩量回调至支撑 | **可以建仓**（10-20%）     |
| 震荡趋势 | 放量突破       | **积极建仓**（30%）        |

### 第四步：止损价计算

```
1. 固定止损：买入价下跌8%止损
2. 均线止损：收盘价跌破20日均线止损
3. 形态止损：跌破关键支撑位止损

建议取三者中最保守的价位作为止损价
```

### 第五步：目标价计算（点图法）

#### Point and Figure Chart 方法

1. **确定箱体大小**：取近期 ATR（平均真实波幅）的1倍或收盘价的1%
2. **画图规则**：

   - 上涨用X表示，下跌用O表示
   - 需要3格转向才能改变列（3格原则）
3. **目标价计算**：

   **方法一：突破目标位**

   ```
   目标价 = 突破点 + (最近列的X格数 × 箱体大小 × 2)
   ```

   **方法二：趋势线目标**

   ```
   目标价 = 当前位置 + (历史最高列的X格数 × 箱体大小)
   ```
4. **简化计算**：

   ```
   短期目标 = 当前价 × 1.10（上涨10%）
   中期目标 = 当前价 × 1.20（上涨20%）
   长期目标 = 当前价 × 1.50（上涨50%）

   根据趋势强度调整：
   - 强势股：短期15%，中期30%，长期60%
   - 弱势股：短期5%，中期10%，长期20%
   ```
